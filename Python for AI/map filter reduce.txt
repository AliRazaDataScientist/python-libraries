map => It applies a function to each element in a sequence and returns a new sequence containing the tansformed elements.It is higher-order function.


filter => It filters a sequence of elements based on a given predicate(a function that returns a boolean value) and return a new sequence containing only the elements that meet the predicate.

reduce => It is a higher-order function that applies a function to a sequcence and return a single value. It is a part of functools module in python and also we should to import reduce otherwise it will throw error
